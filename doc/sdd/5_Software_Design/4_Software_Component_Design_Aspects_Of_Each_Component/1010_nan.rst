nan

.. c:autodoc:: ../libm/mathd/nand.c

Mathematical Approach
^^^^^^^^^^^^^^^^^^^^^

The approach is to return ``0x7FF80000000D067D`` as ``double`` (float: ``0x7FCF067D``). This is a quiet ``NaN`` with the fixed payload ``D067D`` (float: ``F067D``).
